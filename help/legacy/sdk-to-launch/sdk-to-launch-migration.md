# 從獨立 Media SDK 移轉至 Adobe Launch

>[!NOTE]
>Adobe Experience Platform Launch 已經過品牌重塑，現在是 Experience Platform 中的一套資料收集技術。 因此，所有產品文件中出現了幾項術語變更。 如需術語變更的彙整參考資料，請參閱以下[文件](https://experienceleague.adobe.com/docs/experience-platform/tags/term-updates.html?lang=zh-TW)。

本檔案主要著重於Media Analytics實作差異
介於獨立Media SDK和Media Analytics Launch擴充功能之間。

Launch可讓客戶透過簡單的方式部署及管理所有分析，
行銷和廣告解決方案為相關客戶提供支援
在網頁和行動裝置上的體驗。 Launch可供任何使用者建置和維護
自行整合Launch。 這些整合功能又稱「擴充功能」。
Launch客戶可在應用程式商店中取得擴充功能，因此他們
可快速安裝、設定和部署其解決方案。

網頁和行動裝置皆可使用 Launch 擴充功能 Adobe Media Analytics for Audio and Video：

* [Web Launch檔案](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/adobe/media-analytics/overview.html)
* [行動Launch檔案](https://developer.adobe.com/client-sdks/documentation/adobe-media-analytics/)
