# 偵錯控制面板和報表{#debug-dashboards-and-reports}

Adobe Debug提供即時報告，因此您可以檢視視訊播放期間傳送的點選和中繼資料。 這些報表都可儲存在Debug中。

如需認證，您只需複製URL並傳送連結（例如在ZenDesk票證中）即可共用這些報告。

>[!NOTE]
>
>一次只能有一個作用中的工作階段。您可以從控制面板開啟作用中的工作階段。
