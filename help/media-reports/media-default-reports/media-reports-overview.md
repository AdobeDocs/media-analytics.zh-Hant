# 媒體概述{#media-overview}

「媒體概述」控制面板的設計可讓您監視網站上的各個媒體。「媒體概述」畫面會顯示數個彙總測量，供您快速監測媒體是否如預期般執行。圖形會在廣告開始旁顯示內容開始，讓您快速檢視每個媒體項目的這些量度。

![](assets/media_overview.png){width="672px"}
