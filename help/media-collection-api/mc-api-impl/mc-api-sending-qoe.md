# 傳送 QoE 資料{#sending-qoe-data}

您可以使用稱為 `qoeData` 的額外 JSON 索引鍵為每個事件裝飾，將其放置於 JSON 要求內文中的 `params` 索引鍵旁。

>[!NOTE]
>
>請務必參閱 [JSON 驗證結構](/help/media-collection-api/mc-api-impl/mc-api-validate-reqs.md)以確認參數類型，以及瞭解參數為強制或選用性質。
